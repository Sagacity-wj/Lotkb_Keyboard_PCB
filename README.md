# GT_BLE60_Keyboard_PCB
A 60% bluetooth keyboard PCB by Kicad, Schematic fork of https://github.com/Lotlab/BLE60-keyboard  
一个采用Kicad设计制作的60%蓝牙键盘PCB，原理图参考Lotlab的蓝牙键盘，并基于GH60制作。
